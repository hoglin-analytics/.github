MCMetrics is a powerful analytics platform designed for Minecraft servers. It gives you clear, actionable insights into your players, helping you grow your server, improve retention, and make smarter decisions backed by real data.

## Get Started
- Sign up: https://app.mcmetrics.net/auth/signup
- Documentation: https://docs.mcmetrics.net
- Plugins: https://github.com/hoglin-analytics/MCMetrics-Plugins

## What is MCMetrics?
MCMetrics collects anonymized data from your Minecraft server and turns it into meaningful analytics, including:

- Player activity & engagement trends
- Retention and churn insights
- Session tracking
- Server performance metrics
- Custom event tracking (via plugins)

Whether you're running a small community or a large network, MCMetrics helps you understand what’s actually happening on your server.

## Plugins & Integration
To connect your Minecraft server to MCMetrics, you’ll need to install one of our plugins:

https://github.com/hoglin-analytics/MCMetrics-Plugins

These plugins handle:

- Secure data transmission to MCMetrics
- Lightweight performance impact
- Easy configuration

Supported platforms include:
- Velocity
- Bungee
- Paper
- Folia
- (More platforms are being added)

## How It Works
1) Install the plugin on your server
2) Configure your API key from the MCMetrics dashboard
3) Start your server
4) View analytics in real-time via the dashboard

For full setup instructions, visit the docs: https://docs.mcmetrics.net

## Why Use MCMetrics?
- Understand your players - Know when they join, leave, and what they do
- Improve retention – Identify why players stop coming back
- Make data-driven decisions - Stop guessing what works
- Lightweight & scalable – Built to handle servers of all sizes
### Privacy & Data
- Only anonymized gameplay data is collected
- No personal data is exposed publicly
- Data is securely transmitted and stored
### Use Cases
- Server owners optimizing gameplay loops
- Networks tracking player engagement across modes
- Developers testing new features and updates
- Monetization analysis and player behavior tracking
